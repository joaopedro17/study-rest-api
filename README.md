## Endpoints

This is my study on a simple RESTfull API

- Get all posts: ```GET /api/posts```
- Get a single post: ```POST /api/posts/{id}```
- Update a post: ```PUT /api/posts/{id}```
- Delete a post: ```DELETE /api/posts/{id}```
